Register

1. on click "Register" button open modal "RegisterModal" component.

2. when the user fill the form fields, set the state of the field.

3. after the click on "Register" button test the validation of the form fields and show error.

4. send the data to the server. 

5. if no errors or validation errors show page with message that email sent to the user mail with "ok" button.

6. when the user click on "ok" button or "cancel" button, close the modal.
