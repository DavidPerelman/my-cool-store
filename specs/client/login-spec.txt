Register

1. on click "Login" button open modal "LoginModal" component.

2. when the user fill the form fields, set the state of the field.

3. after the click on "Login" button test the validation of the form fields and show error.

4. send the data to the server. 

5. if no errors or validation errors close the modal, hide the login button and show logout button.

6. set the "loggedIn" in "AuthContext" to "true".
